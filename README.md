Welcome to the **Bus Booking App**, a simple web application designed to demonstrate the basic flow of an online bus ticket booking system.

This project allows users to view available routes, book tickets by selecting a route and travel date, and access service-related information. It is created as a learning project to practice frontend development and understand the structure of a booking-based application.

---

## Overview

The application provides a user-friendly interface for:
- Viewing available bus routes
- Booking tickets through a simple form
- Accessing contact and service information

This project focuses on simplicity and clarity rather than advanced functionality.

---

## Technologies Used

- HTML  
- CSS  

---

## Purpose

The main purpose of this project is to:
- Practice web page structuring and form handling
- Understand basic booking application flow
- Build confidence in frontend development
